# 她的职业命运图鉴 · 三力觉醒

一款专为女性职场人设计的互动式职业模拟游戏。通过选择驱动的叙事体验，帮助玩家理解职业发展中「专业力」「硬通货力」「本性力」三大核心能力的动态博弈，直面职场卡点，找到属于自己的破局路径。

## 核心玩法

玩家从自我评估开始，经历四个职业阶段，每个阶段面临真实的职场抉择：

| 阶段 | 时间线 | 核心命题 |
|------|--------|----------|
| 立足期 | 0–2 年 | 生存第一，建立专业基本盘 |
| 成长期 | 2–5 年 | 晋升争夺，突破玻璃天花板 |
| 瓶颈期 | 5–8 年 | 命运岔路，激活真实方向感 |
| 突破期 | 8–15 年 | 构建壁垒，三力整合定位 |

## 三力模型

- **🔧 专业力 (P)** — 岗位特定的技能、知识、经验积累
- **⚡ 硬通货力 (H)** — 执行力、逻辑、目标管理、影响力、沟通
- **🌸 本性力 (N)** — 天赋偏好、内在动力、真实志趣契合度

每次选择都会影响三力数值，最终决定你的职业结局（共 7 种）。

## 职业结局

| 结局 | 触发条件概述 |
|------|-------------|
| 🏆 全栖达人 | 三力均衡且总分极高 |
| 🧠 行业顶尖专家 | 专业力突出 |
| 🎙️ 组织力量者 | 硬通货力突出 |
| 🌬️ 空想怀才者 | 本性强但落地力不足 |
| 💀 高薪内耗者 | 外部成功但本性严重错位 |
| 💔 带伤创业者 | 本性未稳就冲进创业 |
| 🌀 持续进化中 | 均衡但缺乏突出壁垒 |

## 快速开始

本项目为纯前端单文件应用，无需安装任何依赖。

```bash
# 直接用浏览器打开
open index.html

# 或使用任意静态服务器
npx serve .
```

## 技术栈

- 纯 HTML / CSS / JavaScript，零依赖
- 单文件架构（`index.html`）
- 响应式设计，适配移动端
- CSS 变量主题系统 + 渐变动画
